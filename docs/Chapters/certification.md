## 9. Certifications and Formal Credentials

### 9.1 INCOSE CSEP

The INCOSE Certified Systems Engineering Professional (CSEP) is the primary internationally recognised SE certification. It requires a combination of SE education, work experience, and passing a written examination covering all aspects of the SE process as defined in the INCOSE SE Handbook. Preparing for the CSEP examination is an excellent way to systematically study SE fundamentals while working toward a recognised credential.

### 9.2 ECSS Training

ESA and various training providers offer courses on ECSS standards. The ESA Academy and ESTEC provide occasional workshops on ECSS requirements engineering, V&V, and software engineering standards. These courses are directly aligned with the standards used at EUMETSAT.

### 9.3 Capella / MBSE Certifications

Obeo and other Capella training providers offer formal Capella training courses. The Eclipse Capella project provides free online learning materials. Formal Capella training certificates are recognised in the European aerospace industry.

### 9.4 Cloud Certifications

AWS Certified Cloud Practitioner or AWS Certified Solutions Architect (Associate) provides structured knowledge of cloud architectures. Given that EUMETSAT uses both AWS and Azure services, cloud certification demonstrates relevant technical knowledge.
